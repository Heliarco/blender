Learned shortcuts:

Tab: Jump between object and edit mode


Both Modes:
G: Grab / mode
R: Rotate
S: Scale


Object Mode:
Shift + A: Add mesh


Edit Mode:
E: General extrude
Alt + E, Extrude menu

S: Scale
S -> Shift + Z: Scale, but not on Z axis

Alt + S: Shrink'

Shrink, will scale based on normals.
Scale, will use the transform pivot point
